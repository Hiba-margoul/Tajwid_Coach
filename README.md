# Tajwid_Coach
Application tuteur intelligent pour la récitation du Quran. Transcrit votre récitation et corrige automatiquement les règles de Tajwīd. En cours de développement
Ce projet est une application interactive en cours de développement qui aide les utilisateurs à réciter le Quran correctement. Grâce à l’analyse audio et à l’intelligence artificielle, l’application :

Transcrit votre récitation en texte arabe.

Détecte les erreurs de prononciation et de Tajwīd.

Fournit un retour immédiat pour améliorer votre récitation.

L’objectif est de créer un tuteur intelligent personnalisé, accessible à tous, pour apprendre et pratiquer le Tajwīd correctement.

⚠️ Note : Le projet est actuellement en cours de développement et certaines fonctionnalités peuvent être incomplètes.
